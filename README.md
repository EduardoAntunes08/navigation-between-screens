Descrição do Projeto

Este projeto tem como objetivo praticar a navegação entre telas em aplicações Android utilizando Navigation Compose.

Funcionalidades

O app possui 4 telas:

- Login
- Menu
- Perfil
- Pedidos

A navegação é feita com NavController, permitindo a transição entre as telas de forma simples e organizada.

Fluxo de Navegação

Login → Menu
Menu → Perfil / Pedidos
Perfil / Pedidos → Menu
Menu → Login

Conceitos Aplicados

- Navigation Compose
- NavController
- NavHost e rotas
- Organização de telas em Compose
